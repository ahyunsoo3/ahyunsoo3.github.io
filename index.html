What I built

Missroot TOEFL

A real app used by real students to prepare for one of the world's most competitive English exams.

Flutter client

Cross-platform app with Reading, Listening, Speaking, Writing, onboarding, score history, statistics, and monetization flows.

  • Shared design system (core_ui package)
  • Google, Apple, email auth via Supabase
  • Adaptive difficulty engine
  • Force-update version gate

Supabase backend

Postgres + Edge Functions + Storage. All server logic as Deno TypeScript edge functions deployed to Supabase.

  • Server-side purchase verification
  • Exchange-rate & currency API functions
  • Storage bucket media path management
  • User profile sync across auth providers

AI integration

Gemini for content generation and AI scoring. Deepgram for speech recognition in Speaking tasks. Automated TTS pipeline.

  • Writing & Speaking scored by Gemini AI
  • Credit-based usage quota control
  • Validated JSONL content generation
  • Inworld TTS for 4 listening task types

Billing & payments

Three payment providers integrated: Google Play, Apple App Store, and Lemon Squeezy. Subscriptions + one-time packs.

  • Webhook idempotency with billing nonces
  • Free, Plus, Pro, Hyper subscription tiers
  • Leaf → AI credit exchange wallet
  • Cross-platform subscription state sync
